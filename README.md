# Features
* Compress image in `./src/img`
* Generate `_svg.scss` files, according svg files in `.src/svg`. eg. `$filenam: "data:image/svg+xml;......."`

# Require Nodejs
``` 
https://nodejs.org/en/download/
``` 

# Install

Execute

``` 
$ npm install -g gulp bower yarn

$ yarn install

$ bower install
```
# Use
```
$ gulp
```
